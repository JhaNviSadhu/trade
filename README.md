## Git Commit Conventions

```dart
- 🎉 :tada: initial commit 🎉
- 🚀 :rocket: [Add] when implementing a new feature
- 🔨 :hammer: [Fix] when fixing a bug or issue
- 🎨 :art: [Refactor] when refactor/improving code
- 🚧 :construction: [WIP]
- 📝 :pencil: [Minor] Some small updates
```
